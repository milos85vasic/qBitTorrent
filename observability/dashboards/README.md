# Grafana Dashboards

This directory contains provisioning config and dashboard JSON for the merge-search observability stack.

| File | Purpose |
|------|---------|
| `merge-search.json` | Main dashboard — active searches, tracker RPS, latency p95, circuit breaker |
| `default.yml` | Grafana dashboard auto-provisioning config |

See `docs/OBSERVABILITY.md` for the full stack overview.
