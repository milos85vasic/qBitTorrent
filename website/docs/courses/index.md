# Courses

Narrated, recordable video courses for qBittorrent-Fixed. Every
course is text on disk — narration in Markdown, shell commands in a
reproducible `demo.sh`, and an Asciinema cast that plays back the
demo in a real terminal. Nothing binary ships in the repository.

See the canonical overview in
[`courses/README.md`](https://github.com/milos85vasic/qBitTorrent/blob/main/courses/README.md)
for recording + playback instructions.

## Tracks

- **[Operator 101: Your First Search](operator.md)** — end-user tour
  from clone to first downloaded torrent. ~6 minutes.
- **[Authoring a new nova3 search plugin](plugin-author.md)** —
  write and install your own tracker plugin. ~8 minutes.
- **[Contributor deep-dive: TDD + rebuild-reboot](contributor.md)** —
  the RED -> GREEN -> rebuild-reboot -> commit cadence CI enforces.
  ~10 minutes.
- **[Security and operations](security-ops.md)** — threat model,
  scanner bundle, credential rotation, Grafana quickstart.
  ~9 minutes.
